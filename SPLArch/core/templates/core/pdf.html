O PISA é um conversor html2pdf usando o ReportLab Toolkit, o HTML5lib e pyPdf. Ele suporta HTML 5 e CSS 2.1 (e alguns CSS 3). Ele é completamente escrito em Python puro por isso é independente de plataforma. O principal benefício dessa ferramenta que um usuário com habilidades de web como HTML e CSS é capaz de gerar modelos PDF muito rapidamente sem aprender novas tecnologias. Fácil integração em estruturas Python como CherryPy, KID Templating, TurboGears, Django, Zope, Plone, o Google AppEngine (GAE), etc (ver pasta 'demo' para exemplos)
